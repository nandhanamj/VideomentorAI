# 🎥 VideoMentor AI

### AI-Powered Video Learning Assistant

VideoMentor AI is an AI-powered application that transforms YouTube videos and local video files into interactive learning resources. The platform helps users overcome language barriers by transcribing video content, translating it into multiple languages, generating summaries and study notes, creating quizzes, and enabling users to interact with video content through an AI chatbot.

## 🚀 Features

- 🎙️ Speech-to-text transcription using Whisper AI
- 🌍 Multilingual transcript translation
- 📝 AI-generated video summaries
- 📚 Automatic study note generation
- ❓ Quiz generation from video content
- 💬 Chat with video using RAG
- 📥 Export transcripts, notes, and summaries
- 📹 Support for YouTube videos and local video files

## 🏗️ Workflow

```text
Video / YouTube URL
        ↓
Transcription
        ↓
Translation
        ↓
Summary
        ↓
Study Notes
        ↓
Quiz Generation
        ↓
Chat With Video
```
